# Personal Portfolio Website

## Project Description
This is a **responsive personal portfolio website** built with **HTML5, CSS3, and JavaScript**.  
It showcases my **skills, projects, and contact information** and follows **modern web design principles**, including semantic HTML, responsive layouts, and accessibility.

---

## Features
- Fully responsive design for **mobile, tablet, and desktop**  
- **Semantic HTML5** structure (`header`, `nav`, `main`, `section`, `footer`)  
- **CSS Flexbox and Grid** layouts  
- Hamburger menu for mobile navigation  
- Contact form with **basic validation**  
- **Hover effects and transitions** for interactive elements  
- Accessible markup with `alt` attributes and `aria-labels`  
- Cross-browser compatibility  

---

## Technologies Used
- **HTML5** – for the structure of the website  
- **CSS3** – for styling and layout, including variables, grid, flexbox  
- **JavaScript** – for interactive elements (hamburger menu)  
- **Git** – for version control  
- **GitHub Pages** (optional) – for live hosting
